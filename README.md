# X Grapher

X Grapher は、Google の「
[Covid-19 Community Mobility Reports](https://www.google.com/covid19/mobility/)」と、気象庁の「[過去の気象データ](https://www.data.jma.go.jp/gmd/risk/obsdl/index.php)」を使ったインタラクティブインフォグラフィックです。
